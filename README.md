# chuClicker

마우스 위치와 클릭 간격을 설정해 자동 클릭을 실행하는 매크로 프로그램입니다.

---

## 실행 전 필수 설정 (Windows Defender 제외)

Windows Defender가 프로그램을 차단할 수 있습니다. 아래 순서대로 제외 설정을 해주세요.

1. **Windows 보안** 앱 열기 (검색창에 "Windows 보안" 입력)
2. **바이러스 및 위협 방지** 클릭
3. **바이러스 및 위협 방지 설정 관리** 클릭
4. 아래로 스크롤 → **제외 항목** → **제외 항목 추가 또는 제거** 클릭
5. **제외 항목 추가** → **폴더** 선택
6. `chuClicker` 폴더 선택 후 확인

설정 완료 후 `chuClicker.exe` 실행하세요.

---

## 사용법

| 단축키 | 기능 |
|--------|------|
| `F7` | 현재 마우스 위치 기록 + 행 자동 추가 |
| `F6` | 매크로 시작 / 종료 토글 |

### 기본 사용 순서
1. `F7`로 클릭할 위치들을 순서대로 기록
2. 각 행의 `ms` 값 더블클릭으로 클릭 간격 조정
3. `F6` 또는 `[시작]` 버튼으로 매크로 실행
4. 다시 `F6` 또는 `[종료]` 버튼으로 중지

### 저장 / 불러오기
- **저장**: 현재 설정을 `.json` 파일로 저장
- **불러오기**: 저장된 `.json` 파일 불러오기

### 단축키 변경
- 시작/종료 키, 위치 기록 키를 원하는 키로 변경 후 **[적용]** 클릭

---

## 참고
- `ms` 값은 소수점 입력 가능 (예: `0.5`, `15`)
- Windows 특성상 실제 최소 딜레이는 약 1~15ms
